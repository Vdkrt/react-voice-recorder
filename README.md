# React Audio Recorder

Простое одностраничное приложение на React для записи звука с микрофона, просмотра уровня сигнала, обработки аудио и сравнения нескольких вариантов записи.

## Что умеет приложение
- Запускать и останавливать запись с микрофона
- Показывать таймер записи и уровень входного сигнала
- Генерировать три аудиодорожки:
  - raw — исходная запись
  - denoised — запись после шумоподавления
  - opus — версия после сжатия через libopus-wasm
- Воспроизводить треки прямо на странице
- Скачивать каждую дорожку как WAV-файл

## Технологии
- React 18
- Vite
- Web Audio API
- MediaDevices API
- libopus-wasm
- @timephy/rnnoise-wasm
- Vitest

## Структура проекта
```text
src/
  App.jsx          # интерфейс приложения, логика записи, UI и отображение треков
  audio.js         # генерация WAV, ресемплинг и обработка аудио
  audio.test.js    # тест для вспомогательной функции ресемплинга
  main.jsx         # точка входа React
  styles.css       # стили интерфейса
package.json       # зависимости и npm-скрипты
vite.config.js     # конфигурация Vite
```

## Запуск локально
```bash
npm install
npm run dev
```

Откройте адрес, который покажет Vite, обычно http://localhost:5173/.

## Проверка
```bash
npm run test
npm run build
```

## Примечание
Приложение использует браузерные API и требует доступа к микрофону. Для корректной работы лучше запускать его на localhost или через HTTPS.
